# AladdinMIDI – MIDI 转 Arduino IO 映射

本项目适用于 **Aladdin Micro** 开发板（完全兼容 **Arduino Micro** 或 **Leonardo**），用于处理 MIDI 消息、控制 PWM 输出，并读取数字输入以发送 MIDI 消息。

---

## 功能特性

- 通过 USB 接收 MIDI 控制变更消息（CC 消息），控制 PWM 输出
- 读取数字输入引脚状态变化，发送 **MIDI 音符开/关（Note On/Off）** 或 **控制变更（CC）** 消息
- 支持编译时宏配置数字输入的工作模式（`NOTE` 或 `CC`）
- 自动将 MIDI CC 值（0–127）映射为 PWM 输出值（0–255）
- 支持 **6 个 PWM 输出引脚**（Y0–Y5）与 **8 个数字输入引脚**（X0–X7）
- 可选串口调试信息输出

---

## 硬件要求

- **主控板**：**Aladdin Micro** 或 Arduino Micro / Leonardo（需支持原生 USB）
- **输出设备**（可选）：LED 或其他 PWM 控制设备
- **输入设备**（可选）：按钮、开关或 NPN 工业传感器（连接至 X0–X7）

---

## 引脚映射

### MIDI CC → PWM 输出（Y0–Y5）

| Aladdin Micro 输出端口 | 对应 Arduino 引脚 | MIDI CC 编号 |
|------------------------|------------------|--------------|
| Y0                     | 9                | 20           |
| Y1                     | 11               | 21           |
| Y2                     | 6                | 22           |
| Y3                     | 10               | 23           |
| Y4                     | 5                | 24           |
| Y5                     | 13               | 25           |

### 数字输入 → MIDI 输出（X0–X7）

数字输入可配置为发送 **MIDI 音符消息** 或 **MIDI CC 消息**。

| Aladdin Micro 输入端口 | 对应 Arduino 引脚 | 音符模式（MIDI 音符编号） | CC 模式（CC 编号） |
|------------------------|------------------|---------------------------|--------------------|
| X0                     | 23 (A5)          | C4  (60)                  | CC 20              |
| X1                     | 22 (A4)          | D4  (62)                  | CC 21              |
| X2                     | 21 (A3)          | E4  (64)                  | CC 22              |
| X3                     | 20 (A2)          | F4  (65)                  | CC 23              |
| X4                     | 4                | G4  (67)                  | CC 24              |
| X5                     | 12               | A4  (69)                  | CC 25              |
| X6                     | 8                | B4  (71)                  | CC 26              |
| X7                     | 7                | C5  (72)                  | CC 27              |

## 使用方法

### 1. 配置工作模式
- 在 Arduino IDE 中打开 `AladdinMIDI.ino`
- 修改 `DIGITAL_INPUT_MODE` 宏为 `NOTE` 或 `CC`，选择数字输入的行为模式

### 2. 上传程序
- 将程序上传至 Arduino Micro / Leonardo 或 **Aladdin Micro**

### 3. 连接设备
- 通过 USB 将开发板连接至计算机
- 计算机应自动将其识别为 **MIDI 设备**

### 4. 开始控制
- 发送 MIDI CC 消息（编号 20–25）即可控制对应 PWM 输出（Y0–Y5）
- 在 X0–X7 输入端口连接开关、传感器等（支持 NPN 或干触点）
  - 引脚状态从 **LOW → HIGH** 时：发送 **音符开（Note On）** 或 CC 值 **127**
  - 引脚状态从 **HIGH → LOW** 时：发送 **音符关（Note Off）** 或 CC 值 **0**

---

## 调试

如需启用串口调试输出：

1. 在 `AladdinMIDI.ino` 文件顶部取消注释 `#define DEBUG`
2. 重新上传程序
3. 打开 Arduino IDE 的 **串口监视器**，波特率设置为 **115200**
